Add teacher assignment relationship to Course entity.

Requirements:

- Add API endpoint to remove teacher assignment: PATCH /courses/{id}/unassign-teacher
- Ensure referential integrity (cannot assign non-existent teacher)
- Allow courses to exist without assigned teacher (nullable field)
- Database migration should preserve existing Course and Teacher data
