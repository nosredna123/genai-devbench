Create a Student/Course/Teacher CRUD application with Python, FastAPI, and SQLite.

Requirements:
- Use Python 3.11+ with FastAPI framework
- SQLite database for persistence
- Three entities: Student, Course, Teacher
- Each entity must have full CRUD operations (Create, Read, Update, Delete)
- Student fields: id (auto-increment), name (string), email (string, unique), enrollment_date (date)
- Course fields: id (auto-increment), code (string, unique), title (string), credits (integer)
- Teacher fields: id (auto-increment), name (string), email (string, unique), department (string)
- RESTful API endpoints for all operations
- Basic error handling and validation
- API should return JSON responses
- Database schema should be created automatically on startup
